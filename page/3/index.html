Create a static blog using Hexo

Install the Hexo CLI

Create the blog

Add a theme

Configure the blog

Edit /blog/_config.yml:

  • In the site section, set title, subtitle, description, author, language, and timezone to valid values.
  • In the Writing section, set new_post_name to :year-:month-:day-:title.md.
  • In the Home page setting section, set index_generator | per_page to 1.
  • In the Pagination section, set per_page to 0.
  • In the Extensions section, set theme to bootstrap-blog.

Tip

Configuring the new_post_name like this may make it easier to find posts in the file system.

Edit /blog/themes/bootstrap-blog/_config.yml:

  • In the Sidebar section, set about_widget_content to a valid value.

Edit the initial post

Edit /blog/source/hello-world.md.

Run the development server

Navigate to localhost:4000.
